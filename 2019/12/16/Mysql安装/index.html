MySql 8.0.16安装教程

在官网下下载MySQL，我这里是在Windows上下载mysql-8.0.16-winx64,安装方法如下：

1.下载MySQL安装包，如图1-1所示；

图 1-1

2.然后在此界面上点击选中图2-1的标注，如图2-1所示，点击下载；

图 2-1

3.点击Download会弹出图3-1，如下图所示；

4.选择自己的安装路径，我放在了D盘；解压当前压缩包，放在当前目录；

压缩包

5.安装到这个位置后，先配置环境变量，点击电脑->属性->高级系统配置->环境变量；先对系统变量进行设置，如图5-1所示；设置完之后点击确定按钮；

图 5-1

6.打开MySQL的安装路径，在mysql-8.0.16-winx64目录下新建一个文件my.ini，如图6-1所示；然后打开该文件，将以下内容复制到my.ini文件中；

图 6-1

注意：将一下内容复制到my.ini中，然后修改mysql的安装目录，更重要的是创建一个数据库的数据的存放目录，我这里创建的是Datebase,该目录实在D:\MySQL下创建的。如图所示：

图 6-2

图 6-3

7.通过cmd命令，是管理员身份运行，进入到MySQL的安装路径;

8.进入以后，使用命令mysqld –initialize –console,会出现以下结果；

注意：此时有登录密码，记录此时的密码，等到登录时用得到，当然也可以修改密码，该步骤后续有介绍。

9.然后使用命令mysqld install，我这里已经安装过了。

10.输入命令net start mysql，会出现如图所示的结果；

11.然后再输入mysql -u root -p命令，会出现Enter password:这个时候你就需要把第8步记录的密码输入进去，结果如图所示；然后就会出来mysql>的提示；

如果你觉的刚才那密码太难搞，你可以修改密码，在第11步之后输入命令alter user root@localhost identified by ‘111’; 若出现OK，则说明你修改成功。

然后在命令提示符后一直有一个mysql>加一个闪烁的光标在等待，这时候输入exit或quit退出就行了。

  1. 关闭MySql服务

    net stop mysql

MySQL的安装过程到这里就安装成功了！如有不足之处，请大家指出来。希望能够解决大家在安装MySQL遇到的问题。
